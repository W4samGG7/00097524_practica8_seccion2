Respuestas a preguntas de la guia de laboratorio 8:

Pregunta 1-¿Cual es la diferencia entre autenticación y autorizacion?

R//La autenticacion es para identificarte con las credenciales que tenes como pasa en el endpoint de /signin, en cambio, la autorizacion es ver que permisos tiene un usuario ya autenticado y a que puede acceder como en el endpoint /protected.

Pregunta 2-¿Cual es la función del token JWT en la guía?

R// El token JWT es como una clave de acceso digital para un usario no tenga que enviar su contraseña en cada solicitud al servidor, el token JWT le dice al servidor que el usuario ya se autentico
